###                                                PCB技巧（四）

1、贴片元件的封装要单层，插件元件封装要画对。

2、所以元件封装都在顶层，不要画成底层的封装！！！

今天就犯了这个错误，找了半天原因。我想要把这个元件直接放到底层，省的在PCB上设置，直接把元件的封装画成底层不是更省事一点，结果板子到了，直接懵了?! 元件的1脚和电路板对不上。
